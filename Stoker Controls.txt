*****Operator Control*****

X - Run Wall Shot
Y - Run Long Shot

A - Run Intake
B - Run Climber
LS - Engage Rope

DPadUp - Gear Intake Up
DPadDown - Floor Gear Intake (Auto Rise)
LS Button - Floor Gear Intake

LBumper - Hopper Forward
RBumper - Hopper Reverse

Back - Climber EStop
Start - Shooter/Hopper EStop

*****Driver Control*****

LT -Reverse GTA Drive
RT - Forward GTA Drive

X - Score Gear (Deadman)
Y - Run Climber

A - Toggle Intake

Back - Calibrate Gyro
